How it Works

  1. Home Page will have 3 options - Add Code, Redeem Code & Prizes List. On home page, we will also need to show how to and about campaign.
  2. On clicking Add code, user will be taken to a page where they can add the code. To add code user will enter mobile number and code on Paint bucket. If code is correct, they will see success message otherwise they will see error message.
  3. On clicking Redeem code, user will be taken to a page where they can redeem the code. Redeem code page will have list of all items. To redeem an item, user will click Redeem and they will be shown a pop up. User will need to enter their mobile number and click submit. A code will be sent to user’s mobile number and they will have to enter the code to redeem the gift. If code is correct and user has enought points, they will see success message otherwise they will see error message.
  4. Prizes list will show all the prizes a user can redeem.
